# Инструкции для командной работы

## Общие указания для лидеров

Лидеры, после выбора командиров передайте им следующие инструкции для взаимодействия с исполнителями.

---

## Часть 1: Выбор и распределение заданий

### Для лидеров:
- Проанализируйте содержимое веб-страницы с математическими упражнениями:  
  **https://www.math-exercises.com/new/university-and-college-math-exercises-math-problems-and-math-tasks**
- Найдите на странице разделы, посвященные следующим темам:
  - Матрицы (Matrices)
  - Комплексные числа (Complex numbers)
  - Векторы (Vectors)
  - Системы линейных уравнений (Systems of linear equations)
- Соберите прямые ссылки на эти разделы и отправьте их своим командирам

### Для командиров:
- Выберите из предоставленных ссылок упражнения сбалансированного объема ("ни много, ни мало")
- Обеспечьте достаточное покрытие материала без перегрузки команды
- Распределите выбранные упражнения между исполнителями

---

## Часть 2: Выполнение заданий и проверка

### Для исполнителей:
- Выполняйте назначенные упражнения **вручную, на бумаге, карандашом или ручкой**
- Каждое решение должно содержать **подробное объяснение каждого шага**
- Отправьте выполненные упражнения командирам согласованным способом (сканы/фотографии)

### Для командиров:
- Внимательно проверьте полученные работы
- Соберите только правильно выполненные и хорошо объясненные упражнения
- Верните на доработку упражнения с ошибками или недостаточными объяснениями

---

## Часть 3: Отчетность и оценка

### Для командиров:
- Соберите все окончательно проверенные упражнения от исполнителей
- Подготовьте итоговую оценку (квалификацию) работы каждого исполнителя
- **В среду** физически представьте на бумаге собранные и проверенные упражнения

### Для лидеров:
- Получите итоговые оценки от командиров до среды
- **Действия в GitHub:**
  - Внесите полученные оценки исполнителей в соответствующую таблицу в репозитории
  - Поставьте оценку работе командиров на основе:
    - Качества собранных работ
    - Усилий по проверке
